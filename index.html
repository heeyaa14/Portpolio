About Us

끈기와 열정을 갖춘 지원자 박경희입니다!

컴퓨터공학과 졸업 후 부족한 전문지식을 보충하고 더 다양한 것들을 접해보기 위해 학원교육을 받았고 여러 프로젝트를 진행하며 실질적인 감을 익힐 수 있었습니다. 제가 기획하고 구현한 서비스가 사용자들에게 어떤 영향을 미칠지가 기대되고 이 과정을 팀원들과 함께 하나씩 만들어나가는 과정이 즐겁고 설레였습니다. 저는 주로 JAVA, Spring Framework를 이용해서 프로젝트를 진행하였는데 최근에는 데이터수집과 인공지능분야에도 관심이 생겨 Python도 공부 중입니다. 다방면으로 유능한 인재가 되어 회사의 발전에 기여하는 직원이 될 수 있도록 노력하고 있습니다. 현실에 안주하지 않고 무궁무진하게 발전해나가는 제 모습을 지켜봐주십시오!

TECH SKILLS

제가 배운 것들 입니다.

JAVA

JAVA의 기초문법을 익히고 웹사이트, 안드로이드 앱개발 등의 여러 프로젝트에서 다양하게 활용하였습니다.

Spring Framework

springboot를 이용한 Spring starter project 프로젝트와 Spring legacy project도 사용해 본 경험이 있습니다.

JSP/Servlet

MVC패턴을 활용하여 서버와 클라이언트간에 데이터를 주고받는 방식을 프로젝트구현을 통해 이해하였습니다.

Javascript

Jquery를 이용한 비동기통신방법인 Ajax를 사용해보았고 node.js, react.js를 활용한 몇가지 예제도 접해봤습니다.

Application

안드로이드 스튜디오와 이클립스를 사용하여 웹과 앱을 연동시켜 간단한 게임, 이미지게시판 애플리케이션을 구현해봤습니다.

HTML/CSS

태그, 폰트, 색상, 효과, 하이퍼링크 등의 Style 적용법을 익혀 원하는 디자인으로 기본 홈페이지를 제작해봤습니다.

MySQL/Oracle

여러 프로젝트에서 DB를 관리하는 RDBMS를 활용함으로써 관계형 데이터베이스 관리시스템를 이해하게 되었습니다.

GitHub

팀프로젝트를 진행할 때 팀원들 각자 branch를 생성하여 수정사항 전달과 협업작업을 더 효율적이고 간편하게 할 수 있었습니다.

Python

csv파일로부터 데이터를 가져와 추출해보고 numpy, pandas, matplotlib, networkX 등의 라이브러리를 활용해봤습니다.
